# **App Name**: SecuraWatch

## Core Features:

- Global Operations Dashboard: Provide a comprehensive overview of key security metrics including patrols (rondas), incidents, and supervisions in a clear, interactive format.
- Interactive Operational Map: Display real-time operational presence, security patrols, and incident locations on an interactive map, enabling quick visual assessment.
- Personnel and Patrol Management: Functionality to manage security personnel profiles, assign patrol routes and schedules, and track their daily activities.
- Incident Reporting and Auditing: Allow users to log new security incidents with essential details, and review/audit a historical log of all reported incidents.
- Supervision Tracking and Review: Enable the tracking of supervision activities and provide tools for reviewing completed supervisions and generating reports.
- AI Incident Prioritization Tool: Utilize an AI tool to analyze incoming and historical incident data to automatically suggest priority levels or potential risk assessments for new events.

## Style Guidelines:

- Background color: Deep Midnight Grey (#18181B) to establish a sophisticated and serious tone for a security application, consistent with a dark user interface.
- Primary accent color: Golden Yellow (#E5A726), a warm, bright color that provides excellent contrast against the dark background, used for highlighting active elements, important data, and calls to action, evoking a sense of vigilance and attention.
- Secondary accent color: Deep Burnt Orange (#994428), an analogous color to the primary yellow, used for secondary indicators, warnings, or less critical but still important UI elements, complementing the overall serious yet functional aesthetic.
- Headline and Body text font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance, which is ideal for data-intensive dashboards and professional operational displays.
- Use sharp, clean line icons that are minimalistic and instantly recognizable, maintaining a professional and data-focused aesthetic consistent with the security monitoring theme.
- Adopt a clean, two-column layout with a fixed dark sidebar for navigation and a main content area optimized for dashboard components and interactive maps, ensuring logical flow and data accessibility.
- Implement subtle transition animations for sidebar navigation and dynamic data updates, providing a smooth and modern user experience without distracting from critical information.
